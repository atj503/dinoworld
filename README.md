# DinoWorld

A fast-paced 3D platformer built with Three.js. Collect rings while avoiding enemies!

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Deployment

This game is configured for deployment on Vercel:

1. Install Vercel CLI:
```bash
npm install -g vercel
```

2. Deploy:
```bash
vercel
```

## Controls

- WASD: Move
- Space: Jump
- Goal: Collect all rings before time runs out!
- Avoid: Red enemy cubes

## Features

- 3D movement and jumping mechanics
- Collectable rings
- Two types of enemies (patrol and chase)
- Timer and ring counter
- Game over conditions
